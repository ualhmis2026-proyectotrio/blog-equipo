# Introducción al Proyecto

**Fecha:** 5 de marzo de 2026
**Autores:** Carmen María Peña Sevilla, Jesús Montoya Ripoll y David Ureña Arróniz

## Descripción del Proyecto

En este proyecto, nuestro equipo está desarrollando un blog estático utilizando **Jekyll** y desplegándolo en GitHub Pages y Azure. El objetivo principal es aprender sobre el desarrollo colaborativo, el uso de control de versiones con Git, y el despliegue de aplicaciones web de forma conjunta.

## Nuestro Equipo

El equipo encargado de dar vida a este blog y gestionar el repositorio está formado por:
- Carmen María Peña Sevilla
- Jesús Montoya Ripoll
- David Ureña Arróniz

## Objetivos

- Crear un blog estático funcional y rápido con Jekyll.
- Colaborar de forma organizada utilizando Git, GitHub Flow y Pull Requests.
- Desplegar el blog correctamente en GitHub Pages.

## Estructura del Blog

El blog contará con las siguientes secciones:
- **Página principal** con las últimas entradas que vayamos publicando.
- **Páginas individuales** de presentación para cada miembro del equipo (Carmen, Jesús y David).
- **Entradas de blog** sobre diversos temas que iremos aprendiendo y aplicando a lo largo de la asignatura.

## Herramientas Utilizadas

- **Jekyll:** Nuestro generador de sitios estáticos (SSG).
- **GitHub:** Para el control de versiones y trabajo en equipo.
- **Visual Studio Code:** Como editor de código principal.
